Визуальный симулятор скорости движения фрезы.  
Точка движется по линейке с заданной скоростью. Позволяет наглядно оценить разницу между мм/сек и мм/мин.

## 🚀 [Смотреть](https://volkovhl.github.io/Feed-Rate-Visualizer/)
- 📏 Линейка 200 мм с четкой градуировкой (0, 50, 100, 150, 200 мм).
- ⏱ Переключение единиц скорости: **мм/сек** и **мм/мин**.
- 🔴 Красная точка движется вдоль линейки.
- 🔄 Автоматический сброс точки при достижении правого края.
- 🖥 Адаптивный дизайн, работает на телефонах и компьютерах.

## 🛠 Использование
1.  Введите скорость в поле ввода.
2.  Выберите единицы: мм/сек (быстро) или мм/мин.
3.  Наблюдайте за движением точки.
4.  Кнопка "вернуть в 0" сбрасывает позицию в начало.

## 📦 Установка / Запуск локально
Всё работает в одном HTML-файле. Никаких серверов и зависимостей.
1. [Скачать](https://github.com/volkovhl/Feed-Rate-Visualizer/archive/refs/heads/main.zip)
